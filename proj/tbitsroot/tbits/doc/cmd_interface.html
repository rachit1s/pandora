Command Interface

tBits provides command interface to do following three things:

  1. View Article
  2. Update Article
  3. Add Article

View Article

The syntax of to view articles is as follows:

java -Dapp.name=tbits -Dtbits.home="%TBITS_HOME%" -Dhome="%HOME%" transbit.tbits.Helper.Wrapper -ba <System Prefix> -n <Article Id> -a <Ascending Order> [Optional, Default] -d <Descending Order> [Optional]

NOTE: -ba is optional if TBits_BA environment variable is defined.

 

 

Update Article

The syntax is as follows:

java -Dapp.name=tbits -Dtbits.home="%TBITS_HOME%" -Dhome="%HOME%" transbit.tbits.Helper.Wrapper -update -request RequestID -ba BusinessAreaPrefix -category Category -severity Severity -status Status -requestType RequestType -logger Logger -duedate yyyymmddhhMMss -duetime HHmm -subject Subject -description Description -notify 1/0 attachments <file_locations> field1=value1 field2=value2... user_id <user login> exit_app_on_closed_xxxxx <true|false>

NOTE:

  1. Business Area Prefix and RequestID are mandatory.
  2. field1, field2 are the extended field or any other field
  3. user_id is the user on whose behalf you are updating article
  4. exit_app_on_closed_xxxx is a flag that will make it force exit. If you dont mention it, you will have to press CTRL+C to close the command
  5. attachments can be a number of comman separated file locations. It will delete the file and put it in the system

Add Article

java -Dapp.name=tbits -Dtbits.home="%TBITS_HOME%" -Dhome="%HOME%" transbit.tbits.Helper.Wrapper -add -ba BusinessAreaPrefix -category Category -severity Severity -status Status -requesttype RequestType -logger Logger -duedate yyyymmddhhMMss -subject Subject -description Description -duetime HHmm -notify 1/0 attachments <file_locations> field1=value1 field2=value2... user_id <user login> exit_app_on_closed_xxxxx <true|false>

NOTE:

  1. Business Area Prefix and Subject are mandatory
  2. field1, field2 are the extended field or any other field
  3. user_id is the user on whose behalf you are updating article
  4. exit_app_on_closed_xxxx is a flag that will make it force exit. If you dont mention it, you will have to press CTRL+C to close the command
  5. attachments can be a number of comman separated file locations. It will delete the file and put it in the system
